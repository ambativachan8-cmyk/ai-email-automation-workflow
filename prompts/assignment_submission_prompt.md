# Assignment Submission Prompt

You are helping draft an assignment submission email for a hiring process.

## Inputs

- Recipient name: `{recipient_name}`
- Company: `{company}`
- Role: `{role}`
- Assignment name: `{assignment_name}`
- Submission link or attachment note: `{submission_details}`
- Key points to mention: `{key_points}`

## Instructions

Write a clear, professional assignment submission email.

The email should:

- Thank the recipient for the opportunity.
- State that the assignment is complete.
- Mention the submitted artifact or link.
- Briefly summarize what the submission includes.
- Offer to answer questions or walk through the work.
- Keep the tone confident and concise.

End with:

`Regards, Vachan`
